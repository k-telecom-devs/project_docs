# Запись данных с утройства, которые отправляются через MQTT

Для того, чтобы какие-либо поля устройства записывались в БД требуется заполнить данное поле в его типе.

<Info>Поле имеет ключ `majorFields` и принимает в себя массив строк</Info>

Вот один из примеров заполнения данного поля:

```json 
"majorFields": [
    "current_heating_setpoint", "local_temperature", "system_mode", "heat"
]
```

В данном случае записываться в бд будут следующие поля: current_heating_setpoint, local_temperature, system_mode, heat, каждое из которых будет в сообщении в MQTT.
В джсоне получаемом из запроса мы будем видить следующую структуру:
```json
"data": [
    {
        "value": {
            "heat": "OFF",
            "system_mode": "heat",
            "local_temperature": 28,
            "current_heating_setpoint": 13
        }
    }
]
```
Как понятно из примера, данные содержаться в поле `data`.
