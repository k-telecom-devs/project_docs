# Настройка FrontView у устройства занчения

<Warning>Ключ **category** в параметре **config** будет всегда иметь значение **'value'**</Warning>


## Поле "keys":

Ключ каждого объекта соответствует ключу, который мы получаем в MQTT. Например, ключ **state** в json идентичен ключу **state**, который мы получаем в MQTT. 

### Заполнение главного ключа:

```json
"temperature": {
    "number":0,
    "label":"Температура",
    "captionPattern":"{temperature}°C",
    "iconUrl":"https://hs.k-telecom.org/icons/devices/icon_temperature_m.svg",
    "showOnCard":true,
    "hasLimits": true
},
```
