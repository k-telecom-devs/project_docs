# Настройка FrontView у устройства состояния

Данное устройство имеет только состояние и второстепенные параметры

<Warning>Ключ `category` в параметре `config` будет всегда иметь значение `state`</Warning>


## Поле "keys":

Ключ каждого объекта соответствует ключу, который мы получаем в MQTT. Например, ключ `contact` в json идентичен ключу `contact`, который мы получаем в MQTT. 

### Заполнение главного ключа:

```json
"contact": {
    "number":0,
    "values": {
        "true": "normal",
        "false": "warning"
    },
    "captionOfState": {
        "normal": "Закрыто",
        "warning": "Открыто",
        "error": "Ошибка"
    },
    "value": null,
    "label":"Состояние",
    "captionPattern":""
}
```

## Заполнение второстепенного ключа:

```json
"battery": {
    "number":1,
    "label":"Аккумулятор",
    "captionPattern":"{battery}%"
}
```
