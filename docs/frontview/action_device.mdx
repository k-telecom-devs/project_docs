# Настройка FrontView у устройства действия

<Warning>Ключ `category` в параметре `config` будет всегда иметь значение `action`</Warning>


## Поле "keys":

Ключ каждого объекта соответствует ключу, который мы получаем в MQTT. Например, ключ `state` в json идентичен ключу `state`, который мы получаем в MQTT. 

### Заполнение главного ключа:

```json
"state": {
    "number":0,
    "values": {
        "ON": true,
        "OFF": false
    },
    "label":"Состояние",
    "captionPattern":"{state}%"
}
```

## Заполнение второстепенного ключа:

```json
"voltage": {
    "number":1,
    "label":"Напряжение",
    "captionPattern":"{voltage} В",
    "hasLimits": true
},
```
