# Настройка FrontView у умных устройств

Умные устройства имеют уникальный интерфейс, что позволяет не заполнять поле `key` во frontview.

Поэтому у умных устройств мы только заполняем категорию устройства, которая должна соответствовать названию девайс-типа

```json
"config": {
    "category": "Teremok-WIFI"
}
```
